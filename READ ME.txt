Step 1: Open Mysql server using bellow code.

	mysql -u anabig114217 -pBigdata123
	
Step 2: Use Mysql database.
	
	use anabig114217;
	
Step 3: Once your database code get executed then run Mysql script file.
	
	source mysqlscript.sql
		
Step 4: Once Tables are sucessfully created in Mysql then exit from server using bellow code.
		
	quit
	
Step 5: Now import the Mysql table to hdfs for this run the bellow .sh file.

	sh sqoopscript.sh

Step 6: Once Tables are sucessfully imported to hdfs now create tables in hive using bellow code.

	hive -f hive_database.sql

Step 7: Once Tables are sucessfully created on hive lets start the analysis part in impala.

	impala-shell -f impala_analysis_script.sql

step 8: Now run the spark file for EDA and SparkML 

	spark-shell < spark_script.sh
